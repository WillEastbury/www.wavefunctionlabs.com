me / constraints

Shaped by what it cost.

Autism and ADHD-C. Not a label — a set of constraints that shaped the operating model.

Masking is expensive. Years of it. Learning to pass as neurotypical in rooms that weren't designed for how you process. The cognitive overhead of translating every interaction in real-time.

Overload is real. Sensory, social, informational. When the buffer fills, the system degrades. Learning to recognise that before collapse — that's a skill that took decades.

Responsibility doesn't pause for any of it. Family, work, delivery, deadlines. The system has to run regardless of internal state.

This isn't listed for sympathy. It's listed because it's real, and because the capabilities on the other pages have a cost that isn't visible from the output alone.

The need for understanding isn't weakness. It's the minimum viable input for a system that runs this hot.
